# たまゆら

発達特性(ASD・ADHD・LD など)のある子どもの「やること」を、ブラウザ内で動く生成AIが
小さなステップに分解して、1つずつ実行できるように支援するWebアプリです。

**完全無料・アカウント不要・データは端末内(localStorage)のみ・サーバー費用ゼロ**で運用できます。

## できること

- 📝 **タスク入力(テキスト / 音声)** — Web Speech API による音声入力に対応
- ❓ **あいまいなタスクの明確化** — AIが1つだけ質問して内容をはっきりさせる
- 🪜 **タスクの細分化** — 3〜6個の小さなステップに自動分解(1画面1ステップで大きく表示)
- ⏱️ **所要時間のめやす** — 各ステップに「◯ふんくらい」を表示、開始前に合計時間で見通しを提供
- 🤔 **こまったときのQ&A** — 実行中に質問すると、AIがやさしい日本語で短く答える
- ✅ **完了チェック** — ステップごとに「できた!」で進み、全部できたらお祝い
- 📄 **PDF出力** — がんばりの記録(タスクと完了状態)を印刷ダイアログ経由でPDF保存
- 🔊 **読み上げ ON/OFF** — 感覚過敏に配慮し、音声はいつでも切れる(音は読み上げのみ、効果音なし)

## 技術構成

| 項目 | 内容 |
|---|---|
| フレームワーク | Vite + TypeScript(フレームワークなしの軽量SPA) |
| 推論 | [WebLLM](https://github.com/mlc-ai/web-llm)(WebGPU、ブラウザ内実行、API利用料ゼロ) |
| モデル | **Qwen3-0.6B**(q4f16量子化・約500MB)。品質引き上げ時は Qwen3-1.7B に差し替え可 |
| モデルキャッシュ | WebLLM が Cache API に保存(2回目以降の起動は高速) |
| データ保存 | localStorage のみ(外部送信なし) |
| PDF | ブラウザの印刷機能(印刷用CSS)。日本語フォント埋め込み不要で確実 |
| ホスティング | 静的ホスティング(Cloudflare Pages / GitHub Pages / Netlify) |

### フォールバック(かんたんモード)

WebGPU 非対応端末・モデルロード失敗時は、テンプレートベースの「かんたんモード」に自動で
切り替わり、アプリ自体は必ず動作します(片付け・宿題・お風呂などの定番タスクはテンプレート、
それ以外は汎用ステップで分解)。設定画面から手動で切り替えることもできます。

### モデルの差し替え(0.6B → 1.7B)

ブラウザのコンソールで以下を実行してリロードすると Qwen3-1.7B(約1.1GB)に切り替わります:

```js
localStorage.setItem("tamayura.modelId", "Qwen3-1.7B-q4f16_1-MLC");
```

### 小型モデルの出力安定化

- プロンプトでJSON構造を厳密指定+few-shot例を提示
- Qwen3 の思考モードは `/no_think` で無効化し、`<think>` タグは出力から除去
- JSONパース失敗時は形式を念押しして1回リトライ、それでも失敗したらかんたんモードで続行

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # 型チェック + 本番ビルド(dist/)
npm run preview  # ビルド結果の確認
```

## デプロイ

`npm run build` で生成される `dist/` を静的ホスティングに置くだけです。

- **Cloudflare Pages**: ビルドコマンド `npm run build`、出力ディレクトリ `dist`
- **GitHub Pages**: `dist/` を gh-pages ブランチへ(`base: "./"` 設定済みのためサブパスでも動作)

> ⚠️ WebLLM は WebGPU を使うため **HTTPS 配信が必須**です(localhost は例外)。
> モデルの重みは初回に Hugging Face CDN からダウンロードされるため、初回のみネットワークが必要です。

## 対応環境

- WebGPU 対応ブラウザ(Chrome / Edge 113+、Safari 18+ など)+ RAM 4GB 以上を推奨
- 非対応環境では「かんたんモード」で動作
- 音声入力(SpeechRecognition)は Chrome 系で利用可。非対応環境ではマイクボタンが自動的に隠れる

## プライバシー

- 入力されたタスク・記録はすべて端末内の localStorage にのみ保存され、外部に送信されません
- AI推論もすべてブラウザ内で完結します(推論のための通信なし)
- 「せってい」画面からデータを全消去できます
